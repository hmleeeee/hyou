Table Caption

좀 더 자세하고 정확하게 표현하기 위해 strong 안에 테이블의 간략한 제목 기입과
p.ui-caption 태그안에는 자동으로 테이블의 모든 th의 항목텍스트를 기입하여 테이블을 설명합니다.
예) 번호, 제목, 이름, 아이디, 날짜 정보입니다.

Examples

테이블의 제목

title A title B
title C content....

HTML Structure


<caption>
<strong>테이블의 제목</strong>
<p class="ui-caption">[자동으로 테이블의 th의 제목이 들어감]</p>
</caption>

Javascript

$plugins.uiCaption


$plugins.uiCaption();
