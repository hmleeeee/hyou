$plugins.uiSlot

	id:'uiSlot1', 
	current:1, 
	auto:false, 
	single:false 
});
slot 기본 옵션

옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-drop의 아이디명 기입
current: [number] 0 처음 시작 위치
auto: [boolean] false true일 경우 자동실행
single: [boolean] true true일 경우 하나만 노출, false일 경우 걸쳐보이는...
