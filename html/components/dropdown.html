Dropdown

date : 2020.06.10

Dropdown은 기본적으로 토글방식으로 문맥상에서 추가적인 콘텐츠나 메뉴에 사용한다.
기본적으로 12가지 위치를 제공하고 있으며 위치에 따른 자동설정도 가능하다.
크게 두가지로 하나는 문장의 흐름에 맞추어 위치한 경우와 모달팝업과 같이 위치가 밖에 있는 경우가 있다.

Examples

.ui-drop-pnl이 body 바로 안에 있는 구조

z-index 값 상속이나 overflow:hidden 속성으로 body의 자식 태그로 위치한다.

.ui-drop-pnl이 .ui-drop과 같은 영역에 있는 구조

자바스크립트란?

자바스크립트(JavaScript)가 언어로서 지닌 특징에 대해 알아보겠습니다. 이어서 자바스크립트로 무엇을 할 수 있을지, 다른 기술들이 자바스크립트를 어떻게 활용하고 있는지도 이야기해 보겠습니다.

정의

자바스크립트는 ‘웹페이지에 생동감을 불어넣기 위해’ 만들어진 프로그래밍 언어입니다.

자바스크립트로 작성한 프로그램을 스크립트(script) 라고 부릅니다. 스크립트는 웹페이지의 HTML 안에 작성할 수 있는데, 웹페이지를 불러올 때 스크립트가 자동으로 실행됩니다.

스크립트는 특별한 준비나 컴파일 없이 보통의 문자 형태로 작성할 수 있고, 실행도 할 수 있습니다.

이런 관점에서 보면 자바스크립트는 자바(Java)와는 매우 다른 언어라고 할 수 있습니다.

자바스크립트는 브라우저뿐만 아니라 서버에서도 실행할 수 있습니다. 이 외에도 자바스크립트 엔진(JavaScript engine)이라 불리는 특별한 프로그램이 들어 있는 모든 디바이스에서도 동작합니다.

HTML Structure


<button type="button" class="ui-drop" id="uiDrop1"><span>드롭다운</span></button>

<section class="ui-drop-pnl" data-id="uiDrop1">
	<div class="ui-drop-box">
		....
	</div>
	<button type="button" class="ui-drop-close">
		<span class="hide">닫기</span>
	</button>
</section>

CSS Style

.ui-drop-pnl {display:none; position:absolute; z-index:100; }

Javascript

$plugins.uiDropdown


$plugins.uiDropdown({ 
	id:'uiDrop1',
	ps: 'BL',
	hold: true,
	dropSpace: $('body'),
	dropSrc: false,
	dropOffset: false,
	openback:function() { console.log('open callback'); },
	closeback:function() { console.log('close callback'); } 
});
dropdown 기본 옵션

옵션명 유형 기본 설명
id: [string] (필수) ui-drop의 아이디명 기입
ps: [string] 'BL' 총 12가지 위치 설정
  • TL : 상단 좌측정렬
  • TC : 상단 중앙정렬
  • TR : 상단 우측정렬
  • BL : 하단 좌측정렬
  • BC : 하단 중앙정렬
  • BR : 하단 우측정렬
  • RT : 우측 상단정렬
  • RM : 우측 중앙정렬
  • RB : 우측 하단정렬
  • LT : 좌측 상단정렬
  • LM : 좌측 중앙정렬
  • LB : 좌측 하단정렬
hold: [boolean] true 포커스 홀드 여부
  • true : 'hold' 포커스 홀드
  • false : 'sense' 포커스 벗어날 때 자동으로 닫힘
dropSpace: [string][object] $('body') Ajax로 파일을 불러올 위치
'id' or $('#id')
dropSrc: [string] Ajax로 파일을 불러올 url경로
dropOffset: [boolean] false 드롭다운 콘텐츠의 위치에 따라 설정
  • true : 위치값 절대값(offset)
  • false : 위치값 상대값(position)
openback: [function] false 오픈 시 콜백
closeback: [function] false 닫을 시 콜백

$plugins.uiDropdownToggle({
