Components Layout

Floating

date : 2020.06.10

특정위치에서 고정되거나 고정되었던 오브젝트가 원위치로 돌아가는 설정

Examples

상단고정 → 원위치

상단에 fixed로 고정에서 원위치로 들어가는 방식

상단고정 → 원위치1
상단고정 → 원위치2
상단고정 → 원위치3

원위치 → 상단고정

특정위치에 도달하면 원위치에서 벗어나 상단에 fixed로 고정되는 방식

원위치 → 상단고정1


원위치 → 상단고정2


원위치 → 상단고정3

하단고정 → 원위치

하단에 fixed로 고정에서 원위치로 들어가는 방식

하단고정 → 원위치1


하단고정 → 원위치2


하단고정 → 원위치3

원위치 → 하단고정

특정위치에 도달하면 원위치에서 벗어나 하단에 fixed로 고정되는 방식

원위치 → 하단고정1
원위치 → 하단고정2
원위치 → 하단고정3

HTML Structure

basic type


<div class="ui-floating" id="exeFix1">
	<div class="ui-floating-wrap">
		...
	</div>
</div>

CSS Style

base style


.ui-floating-wrap {position:relative;}
.ui-fixed-bottom .ui-floating-wrap {position:fixed; bottom:0; left:0; z-index:999;}
.ui-fixed-top .ui-floating-wrap {position:fixed; top:0; left:0; z-index:999;}

style : netive UI


.ui-floating {border:1px dashed rgb(230, 230, 230);}
.ui-floating-wrap {width:100%;}

Javascript

$plugins.uiFloating

상단고정 → 원위치


// add 로 불러올 요소는 미리 실행이 되야한다.(실행 순서가 중요.)
$plugins.uiFloating({ id:'exeFix12', ps:'bottom', fix:true });
$plugins.uiFloating({ id:'exeFix11', ps:'bottom', add:'exeFix12', fix:true });
$plugins.uiFloating({ id:'exeFix10', ps:'bottom', add:'exeFix11', fix:true });

원위치 → 상단고정


$plugins.uiFloating({ id:'exeFix1', ps:'top', fix:false });
$plugins.uiFloating({ id:'exeFix2', ps:'top', add:'exeFix1' fix:false });
$plugins.uiFloating({ id:'exeFix3', ps:'top', add:'exeFix2' fix:false });

하단고정 → 원위치


// add 로 불러올 요소는 미리 실행이 되야한다.(실행 순서가 중요.)
$plugins.uiFloating({ id:'exeFix6', ps:'bottom', fix:true });
$plugins.uiFloating({ id:'exeFix5', ps:'bottom', add:'exeFix6', fix:true });
$plugins.uiFloating({ id:'exeFix4', ps:'bottom', add:'exeFix5', fix:true });

원위치 → 하단고정


$plugins.uiFloating({ id:'exeFix7', ps:'bottom', fix:false });
$plugins.uiFloating({ id:'exeFix8', ps:'bottom', add:'exeFix7', fix:false });
$plugins.uiFloating({ id:'exeFix9', ps:'bottom', add:'exeFix8', fix:false });
옵션 설명
옵션명 유형 기본 설명
id: [string] (필수) fix follow를 적용 시킬 영역(ui-floating)의 id 명
ps: [string] 'top' 고정될 위치 설정
  • 'top' : 상단
  • 'bottom' : 하단
fix: [boolean] true 고정되어 시작할지 여부
  • true : 고정에서 기본
  • false : 기본에서 고정
add: [string] 고정되어 있는 오브젝트가 이미 있다면 그위치 다음으로 위치하기 위해 해당 영역의 아이디명
callback: [function] function 콜백값으로 아이디와 현재스크롤위치값, 상태변경 위치값
