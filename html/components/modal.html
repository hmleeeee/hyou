Javascript

$plugins.uiModalOpen

content modal option


$plugins.uiModalOpen({ 
	id:'IDname', 
	type: 'normal',
	wrap: $('body'),
	moblieFull: false,
	ps: 'center',
	src: false,
	remove: false,
	modalWidth: false,
	modalHeight: false,
	innerScroll: false,
	mg: 10,
	callback:function() { console.log('open callback'); },
	closeCallback:function() { console.log('close callback'); },
	endfocus:false 
});
$plugins.uiModal 옵션 설명

옵션명 유형 기본 설명
id: [string] (필수) ui-modal의 아이디명 기입
type: [string] 'normal'
  • 'normal' : 콘텐츠 모달
  • 'alert' : 시스템 팝업
  • 'confirm' : 시스템 팝업
wrap: [string][object] $('body') 모달창 코드가 생성될 위치
ps: [string] 'center' 모달 레이어 팝업 나타날 위치 설정
  • 'top' : 상단 위치
  • 'center' : 중앙 위치
  • 'bottom' : 하단 위치
mobileFull: [boolean] false mobile에서 전체 사이즈 모달 레이어 팝업 설정
  • true : 전체화면
  • false : 기본 사이즈
src: [string] Ajax로 파일을 불러올 url경로
remove: [boolean] false 코드 삭제 여부
modalWidth: [number] 모달 레이어 팝업 가로 크기
옵션이 없을 경우 모달 팝업의 크기
modalHeight: [number] 모달 레이어 팝업 세로 크기
옵션이 없을 경우 모달 팝업의 크기
innerScroll: [boolean] false true 모달창 콘텐츠 내부에 스크롤 생성
mg: [number] 10 모달창 상하 여백 크기
callback: [function] false 모달 레이어 팝업 오픈 시 콜백값 실행
closeCallback: [function] false 모달 레이어 팝업 닫을 시 콜백값 실행
endfocus: [string] document.activeElement 모달 레이어 팝업 닫을 시 포커스가 갈 위치의 아이디 값 설정
기본으로 모달 레이어 팝업 띄우기 전 포커스로 설정되나 그 값도 없을 경우 body로 설정

system modal option


$plugins.uiModalOpen({ 
	id:'IDname', 
	type: 'alert', //'confirm'
	ps: 'center',

	sMessage: '',
	sBtnConfirmTxt: 'Ok',
	sBtnCancelTxt: 'Cancel',
	sZindex: false,
	sClass: 'type-system',
	sConfirmCallback: false,
	sCancelCallback: false 
});
$plugins.uiModal - system modal option 설명

옵션명 유형 기본 설명
sMessage: [string] 메시지 입력. 태그가능
sBtnConfirmTxt: [string] confirm 버튼 명
sBtnCancelTxt: [string] cancel 버튼 명
sZindex: [string] zindex 값 지정
sClass: [string] class 추가
sConfirmCallback: [string] confirm 클릭 시 콜백
sCancelCallback: [string] cancel 클릭 시 콜백

$plugins.uiModalClose

모달 레이어 팝업 닫을 시 사용


$plugins.uiModalClose({ 
	id: 'modal_name', 
	remove: false,
	endfocus: 'idname', //$('#idname')
	closeCallback: function(v){console.log(v)} 
});
$plugins.uiModalClose 옵션 설명

옵션명 유형 기본 설명
id: [string] (필수) ui-modal의 아이디명 기입
remove: [boolean] false 모달팝업 콘텐츠 삭제 여부
  • true: 코드삭제
  • false: 미삭제
endfocus: [string] document.activeElement 모달팝업 닫을 시 포커스가 갈 위치의 아이디 값 설정
기본으로 모달팝업 띄우기 전 포커스로 설정되나 그 값도 없을 경우 body로 설정
closeCallback: [function] 모달팝업 닫을 시 콜백함수 실행
