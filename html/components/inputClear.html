Components Form

Input Clear

IE에서 input 태그의 text 속성을 가지는 입력폼을 사용하는 경우 자동으로 생성되는 X버튼 기능구현

Examples

HTML Structure


//value 값이 없는 경우
<input type="text" id="a7" class="inp-base ui-inpcancel">

//value 값 입력 시 생성
<input type="text" id="a7" class="inp-base ui-inpcancel" value="test">
<button type="button" class="ui-inpcancel-btn" data-id="a7"><span>clear</span></button>

CSS Style

base style


.ui-inpcancel-btn {display:inline-block; position:absolute; margin-left:-40px; width:40px; height:40px; }
.ui-inpcancel-btn span {position:relative; font-family:'Material Icons'; display:block; left:10px; width:20px; height:20px; background:transparent; overflow:hidden; font-weight:600; font-size:20px; color:#000;}

Javascript

$plugins.uiInputClear


$plugins.uiInputClear();
