Javascript

$plugins.uiSelect()

현재 select box기준으로 디자인 select 생성. 첫적용이나 동적인 변경이 있을때 사용한다. 특히 동적인 변경일 경우는 id로 개별 실행해주는게 좋다.


//전체 실행
$plugins.uiSelect();

//개별 실행
$plugins.uiSelect({ 
	id: false, 
	current: null,
	customscroll: true,
	callback: false
});

$plugins.uiSelectAction()

마우스, 키보드 이벤트가 아닌 option값 선택을 할 때 사용. $plugins.uiSelect()을 사용해도 되나 불필요한 재구성을 하지 않아 값변경 시에는 $plugins.uiSelectAction를 사용한다. 


$plugins.uiSelectAction({ 
	id:'forSelLocal', 
	current:0,
	callback:function(v){console.log(v)}
});
selectg 기본 옵션

옵션명 유형 기본 설명
id: [string] false .ui-select에 포함된 select의 아이디명 기입
current: [number] null select option 선택
customscroll: [boolean] true custom scroll 적용
callback: [function] false 값 선택 시 콜백실행.
