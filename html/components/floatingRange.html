Floating Range

date : 2020.06.10

특정 범위안에서만 스크롤 이동에 따라 이동하는 오브젝트

Examples

특정 영역 안에서만 이동
특정 영역 안에서만 이동

HTML Structure

basic type


<div class="ui-floating-range" id="uiflRange01">
	<div class="ui-floating-range-item">
		...
	</div>
</div>

CSS Style

base style


.ui-floating-range {position:relative; }
.ui-floating-range-item {position:relative; top:0; left:0;}

style : netive UI


.ui-floating-range {display:inline-block; vertical-align: middle; height:500px; width:300px; border:1px dashed #ccc;}
.ui-floating-range-item {width:100%;}

Javascript

$plugins.uiFloatingRange


$plugins.uiFloatingRange({ 
	id:'uiflRange01', 
	margin: 20
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) fix follow를 적용 시킬 영역(ui-floating)의 id 명
margin: [number] 0 오브젝트 상단영역 마진값 추가로 이동시점 변경
