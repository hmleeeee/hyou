HTML Structure

Radio

Checkbox


// base checkbox
<input type="checkbox" id="c1_1">
<label class="lb-check" for="c1_1"><span>선택</span></label>

// all checkbox (전체체크의 그룹 체크박스에 전체체크의 id값의 class추가)
<input type="checkbox" id="c2_0">
<label class="lb-check" for="c2_0"><span>전체체크</span></label>
<input type="checkbox" id="c2_1" checkgroup="c2_0">
<label class="lb-check" for="c2_1"><span>선택1</span></label>
<input type="checkbox" id="c2_2" checkgroup="c2_0">
<label class="lb-check" for="c2_2"><span>선택2</span></label>

Javascript

$plugins.uiSelection


// 전체실행
$plugins.uiSelection();

// 특정 아이디 실행
$plugins.uiSelection({ 
	id:'name', 
	all:true, 
	callback:function(v){...} 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' 생략 시 기본적인 type="radio" 와 type="checkbox" 전체에 반영
all: [boolean] false 오직 체크박스에서만 사용
  • true : 체크박스 전체 체크 기능을 사용
  • false : 사용안함
callback: [function] function 콜백값으로 아이디와 체크상태 값

$plugins.uiDisabled


$plugins.uiDisabled({ 
	id:'name', 
	disabled:true, 
	callback:function(v){...} 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' 생략 시 기본적인 type="radio" 와 type="checkbox" 전체에 반영
disabled: [boolean] false disabled 상태 설정
  • true : 선택불가상태
  • false : 선택가능상태
callback: [function] function 콜백값으로 아이디와 체크상태 값

$plugins.uiChecked


$plugins.uiChecked({ 
	id:'name', 
	checked:true, 
	callback:function(v){...} 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' 생략 시 기본적인 type="radio" 와 type="checkbox" 전체에 반영
checked: [boolean] false checked 상태 설정
  • true : 체크상태
  • false : 미체크상태
callback: [function] function 콜백값으로 아이디와 체크상태 값
