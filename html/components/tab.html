Javascript

$plugins.uiTab


$plugins.uiTab({ 
	id:'exeTab1', 
	current:0, 
	onePanel: false,
	effect: false,
	align : 'center',
	callback:function(v){...} 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-tab의 아이디명 기입
current: [number] 0 처음 열린패널 선택
onePanel : [boolean] false 탭 선택 시 컨텐츠 변경 여부
  • true : one panel
  • false : multiple panel
effect : [string] false class 추가 (eff-fade)
align : [string] center 가로 스크롤이 있을 경우 선택된 탭의 위치 기준
  • center : 중앙에 위치
  • left : 좌측에 위치
callback: [function] function 콜백값으로 아이디와 현재선택 index값

$plugins.uiTabAction


$plugins.uiTabAction({ 
	id:'exeTab1', 
	current:0, 
	callback:function(v){ console.log(v); } 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-tab의 아이디명 기입
current: [number] 0 열린패널 선택
callback: [function] function 콜백값으로 아이디와 현재선택 index값

parameter 값으로 탭 선택방법


single 선택
/netiveUI/html/components/tab.html?tab=1 

복수 선택
/netiveUI/html/components/tab.html?tab=exeTab1*0+exeTab1_2*1+exeTab2*1 
탭아이디*선택+탭아이디*선택....
주의 : tab의 아이디를 '*'이 들어가는 이름으로 설정하면 안된다.
