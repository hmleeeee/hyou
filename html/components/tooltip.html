Tooltip

툴팁(말풍선)은 마우스오버나 포커스가 가는 즉시 동작하여 보충설명을 보여준다.
항상 상위에 보이기 위해(z-index, overflow:hidden) baseLayer 위치에 생성한다.
기본적으로 자동으로 툴팁 실행위치를 계산하여 더 많은 공간이 있는 위치를 선택하여 보여준다.

HTML Structure


//툴팁 자동생성 하거나 미리 넣어도 됨.
<div class="ui-tooltip" id="tt" role="tooltip" aria-hidden="true">
	....
</div>

//툴팁 버튼 ajax 로 불러오기
<a href="#tt" aria-describedby="tt" class="ui-tooltip-btn" data-class="tip-test0" data-src="/netiveUI/html/Components/tooltip_ajax.html">tooltip</a> 
<button type="button" aria-describedby="tt" class="ui-tooltip-btn" data-class="tip-test0" data-src="/netiveUI/html/Components/tooltip_ajax.html">tooltip</button>
