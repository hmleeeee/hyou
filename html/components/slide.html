Javascript

$plugins.uiSlide


$plugins.uiSlide({ 
	id:'name', 
	current:0, 
	eff:'slide', 
	speed:300, 
	dot:true, 
	nav:true, 
	auto:true, 
	gauge:true, 
	play:false, 
	autoTime:3000, 
	mouseDrag:true, 
	touchDrag:true,
	loop:true, 
	items:1, 
	ulti:false, 
	margin:0 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-slide의 아이디명 기입
current: [number] 0 처음 노출될 아이템 선택
eff: [string] 'slide' 아이템 전환 효과
  • 'slide': 슬라이드 효과
  • 'fade': 페이드 효과
speed: [number] 300 eff 속도
dot: [boolean] true dot navigation 노출여부
nav: [boolean] true next/prev button 노출여부
auto: [boolean] true 자동진행 사용 여부
gauge: [boolean] true 자동진행 시 게이지 노출여부
play: [boolean] false 자동진행 시작 시 플레이 여부 선택
autoTime: [number] 3000 자동진행 간격시간
mouseDrag: [boolean] true pc에서 마우스 드래그 사용여부
touchDrag: [boolean] true 터치 드래그 사용여부
callback: [function] function 콜백값으로 아이디와 현재선택 index값
loop: [boolean] true (미구현) 루프선택
items: [number] 1 (미구현) 보여줄 아이템 갯수
multi: [boolean] false (미구현) 모든 아이템이 나열되어 보여줌
margin: [number] 0 (미구현) multi 사용시 아이템간의 간격

$plugins.uiSlideFnEvt

지정된 아이템으로 전환


$plugins.uiSlideFnEvt({ 
	id:'name', 
	current:0 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-slide의 아이디명 기입
current: [number] 0 노출될 아이템 선택

$plugins.uiSlideFnAuto

자동진행 play/stop 컨트롤


$plugins.uiSlideFnAuto({ 
	id:'name', 
	play:false 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-slide의 아이디명 기입
current: [boolean] 자동진행 play/stop
  • true: 자동진행 플레이
  • false: 자동진행 정지
