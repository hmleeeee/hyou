Json Coding List

IA엑셀 문서 기준으로 만들어진 json파일형식의 엑셀에서 변환한 json파일로 자동 생성 관리
일정관리를 엑셀문서로 진행하여 공유

Examples

코딩 리스트 설명

상태
대기 : 작업 전 (white)
검수 : 작업 완료 / 검수 전 (yellow)
완료 : 작업 완료 / 검수 완료 (violet)
일정
일정 : 작업완료 예정일
완료일 : 작업 완료일
수정일 : 작업 완료 후 수정한 일
배경색
black : 일정지연
skyblue : 예정일정 3일전

코딩 리스트 excel

IA엑셀 문서에서 필요한 정보를 셀을 복사하여 구성한다. excel파일에 IA문서 버전을 기록하면 버전관리에 도움이 된다.

$plugins.uiCodinglist


$plugins.uiCodinglist({ id:'uiCodinglist', url:'/netiveUI/resources/data/codinglist.json', type:'json'});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) 코딩리스트를 적용 시킬 영역(ui-codinglist)의 id 명
url: [string] 'link' (필수) 코딩리스트 정보를 불러올 json or html 파일 경로
type: [string] 'json' 문서타입 설정. 기본으로 json 문서 타입 설정됨
  • 'json'
  • 'html'
