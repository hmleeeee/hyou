Components

Print

특정영역만 프린트를 해야하는 경우에 사용.

Examples

나의 아저씨

점선영역의 이미지와 텍스트를 프린트합니다.


HTML Structure


//실행 전
<body>
	<div print-id="prtSct">
		....			
	</div>
</body>

//실행 후
<body class="ui-print-ing">
	<div print-id="prtSct">
		....			
	</div>
	<div class="ui-print">
		<div print-id="prtSct">
			....			
		</div>
	</div>
</body>

CSS Style

base style


body.ui-print-ing > * {display:none}
body.ui-print-ing .ui-print {display:flex; width:100%; height:100%; justify-content: center; align-items: center; position:fixed; opacity:0.5}
@media print {
	body.ui-print-ing > * {display:none}
	body.ui-print-ing .ui-print { display:block; opacity:1; position: relative;}
}

Javascript

$plugins.uiPrint


$plugins.uiPrint({ 
	id:'prtSct' 
});

※ IE 에서 프린트 미리보기 기능을 사용하기 위한 설정이다.
- 도구 > 인터넷 옵션 > 보안 탭 > 신뢰할 수 있는 사이트 선택
1. 사이트 버튼 클릭 > 사이트 추가
2. 사용자 지정 수준 클릭 > 스크립팅하기 안전하지 않은 것으로 표시된 ActiveX 컨트롤 (사용)으로 체크
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) 프린트할 영역의 print-id
