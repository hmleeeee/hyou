Table Explanation

caption

좀 더 자세하고 정확하게 표현하기 위해 strong 안에 테이블의 간략한 제목 기입과 p.ui-caption 태그안에는 자동으로 테이블의 모든 th의 항목텍스트를 기입하여 테이블을 설명합니다.
예) 번호, 제목, 이름, 아이디, 날짜 정보입니다.


<caption>
<strong>테이블의 제목</strong>
<p class="ui-caption">[자동으로 테이블의 th의 제목이 들어감]</p>
</caption>

col

col 스타일은 클래스로 사용하여 html 마크업과 css 스타일을 분리시킨다.
테이블이 가지고 있는 구분지어지는 클래스 기준으로 n1, n2 ... 로 개별적으로 테이블 col의 width 관련 스타일을 준다. 


<table class="tbl-base type-table">
...
<col class="n1">
<col class="n2">
<colgroup span="2" class="n3">
class 예약어
클래스명 설명
.col-auto 크기가 유동적일때 사용.
width="auto"
.col-[1] 상위 부모의 class 기준(type-info)으로 col-1,2,3 으로 숫자로 설정
예)
.type-name .col-1 {width:100px}
.type-name .col-2 {width:200px}
.col-[size] 기본적인 고정 사이즈로 % 기준으로 설정. 주로 콘텐츠 data형 테이블에 사용
  • .col-xxxs {width:5%}
  • .col-xxs {width:10%}
  • .col-xs {width:15%}
  • .col-s {width:20%}
  • .col-m {width:25%}
  • .col-l {width:30%}
  • .col-xl {width:35%}
  • .col-xxl {width:40%}
  • .col-xxxl {width:45%}
  • .col-half {width:50%}
.col-[name] 게시판형 목록 테이블에서 크기를 일정하게 사용하는 항목에 사용
예)
.col-name {width:100px} /* 이름, 성명 */
.col-date {width:200px} /* yyyy-mm-dd 형의 날짜 */

scope="colgroup" & <colgroup>

scope="colgroup" 열그룹은 colgroup 범위안에서 그룹핑을 한다. 사용 시 colgroup으로 영역 구분.


<colgroup >
	<col class="col-th">
	<col class="col-auto">
	<col class="col-auto">
</colgroup>
<colgroup >
	<col class="col-th">
	<col class="col-auto">
	<col class="col-auto">
</colgroup>

<tr>
	<th scope="colgroup">열그룹 A</th>
	<td>열그룹 A-1</td>
	<td>열그룹 A-2</td>

	<th scope="colgroup">열그룹 B</th>
	<td>열그룹 B-1</td>
	<td>열그룹 B-2</td>
</tr>

scope="rowgroup" & <tbody>

scope="rowgroup" 행그룹은 tbody 범위안에서 그룹핑을 한다. 사용 시 tbody으로 영역 구분.


<tbody>
	<tr>
		<th scope="rowgroup" rowspan="2">행그룹 A</th>
		<th scope="row">행그룹 A-a</th>
		<td>행그룹 A-a-1</td>
	</tr>
	<tr>
		<th scope="row">행그룹 A-b</th>
		<td>행그룹 A-b-2</td>
	</tr>
</tbody>
<tbody>
	<tr>
		<th scope="rowgroup" rowspan="2">행그룹 B</th>
		<th scope="row">행그룹 B-a</th>
		<td>행그룹 B-a-1</td>
	</tr>
	<tr>
		<th scope="row">행그룹 B-b</th>
		<td>행그룹 B-b-2</td>
	</tr>
</tbody>
