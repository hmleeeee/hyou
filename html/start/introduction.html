Before the start

Netive UI Plugins는 jQuery기반으로 작업이 되어있어 jQuery 플러그인을 먼저 실행 후 사용해야 합니다.

HTML Template

HTML5 Doctype를 기본으로 구성한 아래와 같이 작성한다.
특별한 이유가 있지 않다면 스크립트는 body 최하단영역에 위치한다.


<doctype html>
<html lang="ko">
	<head>
		<meta charset="utf-8">
		<meta http-equiv="X-UA-Compatible" content="IE=edge">
		<meta name="viewport" content="width=device-width, initial-scale=1.0, shrink-to-fit=no, user-scalable=yes">

		<title>page title</title>

		<link rel="stylesheet" href="/netiveUI/resources/css/ui.base.css">

		<script defer src="/netiveUI/resources/js/lib/jquery-1.12.4.min.js"></script>
		<script defer src="/netiveUI/resources/js/ui.global.js"></script>
		<script defer src="/netiveUI/resources/js/ui.common.js"></script>
		<script defer src="/netiveUI/resources/js/ui.page.js"></script>
	</head>
	<body>
		<div class="base-skip" id="baseSkip"></div>
		<div class="base-wrap" id="baseWrap">
			<header class="base-header" id="baseHeader"></header>
			<div class="base-body" id="baseBody">
				<aside class="base-aside" id="baseAside">
					// 사이드 영역 
				</aside>
				<main role="main" class="base-main" id="baseMain">
					// 콘텐츠 영역
				</main>
			</div>
			<footer class="base-footer" id="baseFooter"></footer>
		</div>

		
		<script>
		;(function($, win, doc, undefined) {
			$(doc).ready(function(){
				// 스크립트영역  
			});
		})(jQuery, window, document);
		</script>
	</body>
</html>

Z-index

기본적인 z-index 값을 정하여 체계적인 위치설정

옵션 설명
class명 옵션명 설정값 설명
.base-skip z-index 1000 스킵네비게이션 메뉴 (모든 컨텐츠의 최상위에 존재)
.modal-backdrop z-index 199 모달의 암막레이어
.ui-modal z-index 200 ~ 모달
.ui-dropdown z-index 100 드롭다운
.ui-tooltip z-index 50 툴팁
