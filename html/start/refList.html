Brick List

date : 2020.06.09

벽돌을 쌓아 올리듯 비어있는 공간이 없이 정렬시키는 리스트이다.
css의 가로 길이와 옵션의 fixCol 값으로 기준을 주는 두가지 방식으로 사용합니다.

Examples

ui-bricklist-item의 정해진 css width 값 기준으로 정렬

HTML Structure


<div class="ui-bricklist" id="uiBrickList1">
	<ul class="ui-bricklist-wrap">
		<li class="ui-bricklist-item">
			<div><img data-src="/hyou/resources/img/design/@iu1.jpg" alt=""></div>
			<strong>title</strong>
		</li>

		//disabled 클래스 추가 시 리스트에서 제외
		<li class="ui-bricklist-item disabled">
			<div><img data-src="/hyou/resources/img/design/@iu1.jpg" alt=""></div>
			<strong>title</strong>
		</li>
		...
	</ul>
</div>

CSS Style

base style


.ui-bricklist {position:relative; width:100%;}
.ui-bricklist-wrap {margin:0; padding:0;}
.ui-bricklist-item {width:150px; display:block; box-sizing:border-box; overflow:hidden;}
.ui-bricklist-item img {width:100%; display:block}

style : netive UI


.ui-bricklist .ui-bricklist-wrap {background:#ececec;}
.ui-bricklist .ui-bricklist-item {opacity:0; transform:scale(0.8); transition:transform 0.15s ease-in, opacity 0.3s ease-in, top 0.3s ease-in-out, left 0.3s ease-in-out;}
.ui-bricklist .ui-bricklist-item.on {transform:scale(1); opacity:1;}
.ui-bricklist .ui-bricklist-item {width:150px; color:#444; text-align:center;}

Javascript

$plugins.uiBrickList


$plugins.uiBrickList({ 
	id: 'uiBrickList1',
	fixCol: {
		1024 : 5,
		600 : 3,
		400 : 1
	},
	response: true 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-bricklist의 아이디명 기입
fixCol: [object] {1024 : 5, 600 : 3, 400 : 1} width값에 따른 갯수 변경
response: [boolean] true resize 시 재구성여부

$plugins.uiBrickListItem

동적으로 추가되는 리스트가 있을 경우 사용


$plugins.uiBrickListItem({ 
	id:'uiBrickList1'
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-bricklist의 아이디명 기입
